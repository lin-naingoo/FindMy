# FindMy
Simple React game to find an object on the board

- Click on a cell of the board to reveal the value (e.g. 3x3 matrix)
    - 🍏 - get a score
    - 💣 - lose a life
- If goal is reached, you win! 🏆
- If lives are empty, you lose! 😭
